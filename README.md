# Red Fox Tech Consulting Website

Official website for Red Fox Tech Consulting, a student-governed technology consulting society at Marist University.

The website presents the society's mission, services, leadership, events, application process, and contact information. It was built as a static multi-page website using HTML, CSS, JavaScript, and local media assets.

## About Red Fox Tech Consulting

Red Fox Tech Consulting helps local organizations and campus partners solve practical technology problems. The society's work is free and best-effort, with clear scope and clean handoff, so clients receive usable solutions while student members build relevant real-world skills.

## How to Run Locally

Because this is a static website, it can be opened directly in a browser.

1. Download or clone the repository.
2. Open `index.html` in a web browser.
3. Navigate through the site using the menu links.

If you use Visual Studio Code, you can also run the project with the Live Server extension for a smoother local development experience.

## GitHub Upload Notes

Before uploading publicly, confirm that all images, logos, headshots, videos, and Marist-related branding are allowed to be shared in a public repository.

Do not upload:

- Passwords
- API keys
- `.env` files
- Private student information
- Files that should remain internal to the society or university
